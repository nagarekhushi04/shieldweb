# 🎤 ShieldWeb3 — Demo Day Presentation Guide

## 1. The Hook (0-15s)
*   "Phishing is the #1 threat in Web3, and standard tools are too slow."
*   "ShieldWeb3 is a real-time, decentralized anti-phishing layer that combines AI with the trust of the Stellar blockchain."

## 2. The Problem (15-45s)
*   Fragmented security alerts and slow on-chain registries.
*   Billions lost annually to users clicking malicious links.
*   Censorship of threat data by centralized security vendors.

## 3. The ShieldWeb3 Solution (45s-2m)
*   **Decentralized Intelligence:** A global network of researchers rewarded in **SHW3** tokens.
*   **Real-time AI:** Python-based ML classifying domains in milliseconds.
*   **On-Chain Truth:** Every threat report is verifiable on the Stellar ledger.
*   **Seamless Interface:** Chrome extension protection + high-fidelity dashboard.

## 4. Key Metrics & Growth (2m-2m30s)
*   **36 Meaningful Commits:** Rapid feature development and stabilization.
*   **30+ Verified Testnet Users:** A growing network of defenders.
*   **94.2% AI Accuracy:** Proven classification of malicious domains.
*   **Real-time WebSocket Strearm:** Instant threat broadcasting to all users.

## 5. Roadmap & Vision (2m30s-3m)
*   Expanding to Ethereum with a MetaMask Snap.
*   Stellar/Soroban cross-chain state anchoring.
*   Transitioning to a fully DAO-governed threat registry.
